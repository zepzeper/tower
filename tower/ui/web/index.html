Tower API Middleware

Welcome to the API Middleware

This service connects and transforms data between different APIs.

The server is up and running. You can interact with it through the available endpoints below.

API Endpoints

GET /health

Check the health status of the server.

GET /api/v1/channels

Get a list of all available channels.

POST /api/v1/channels

Create a new channel.

GET /api/v1/workflows

Get a list of all workflows.

POST /api/v1/workflows

Create a new workflow.

GET /api/v1/transformers

Get a list of all transformers.

Try it out!

You can use cURL to interact with the API:

curl -X GET http://localhost:8080/api/v1/channels
